SPLAT

Kinect driven data visualization

Access Github Repository »

Kinect motion based navigation

SPLAT is a Kinect Driven data visualization of public transportation deaths and accidents over a ten year span. The program was written in Visual Studio using C# and WPF, and the Microsoft Kinect SDK to navigate. The initial screen is a simple title menu that explains the idea of the project while the Kinect interface loads. Once loaded, a hand icon appears on the screen and can be moved and gestured with the user’s hand. Pushing the screen navigates into the main view. The Main view represents the base class of the program, and all of the possible functionality. Navigating and moving your hand forward on the USA STATS button opens up a new window with a graph of the overall deaths and accidents in the US over the last decade. The regions are all color coded. Hovering over any of the regions makes it pop forward, and pushing in opens a new region window with individual stats and states.

Clear, organized Visualizations

The Region window inherits from the main window and adds to the functionality with a back button on the top left, a Regional Stats button, and individual states. Navigating to different states was too cumbersome on the region itself so the scroll bar proved more efficient and functional. The user can hover over, grab, and drag to scroll across the list of states, or push in to select a state.

Comprehensive Information

State windows inherit from the STATS windows and display graphs as well as buttons to navigate to the main window, or the region in which the state belongs. The graphs show the user recent accident and death data for public transportation for every state
