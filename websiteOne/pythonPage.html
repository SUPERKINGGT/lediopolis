An automated security camera turret powered by the Raspberry Pi and Driven by Python

Easy touch-friendly UI

Mission Statement: Build a two camera security turret that monitors an area and takes surveillance photos when motion is detected. Upload those photos to twitter automatically and alert the home owner view tweet. - Loose stream of thought instructions Materials: Raspberry Pi USB hub – powered Mouse / Keyboard / Wifi Dongle Logitech Webcam Canon SLR What I did: Connect to your Pi – preferably learn to SSH into your Pi for convenience down the road. Open your terminal and download Motion. Motion is fantastic because it simplifies a good deal of the work. The Raspberry Pi with Raspbian isn’t an ideal candidate for motion as higher frame rates and resolutions really strain the processor but for a proof of concept project it does alright. I’ve heard others have had more success with Arch Linux on the Pi but have not tested. Motion will detect the Logitech webcam and automatically run the server. Edit the motion script to react to significant pixel shifts in the video stream and run a python script. Since video is simply a string of photos played in succession, you can have success writing a python script that compares pixel values of consecutive images and detects change as motion.

Practical paint application

TNavigate your browser of choice to Twitter/Dev and register a dev account. Retrieve the appropriate tokens and set them up with Twython. In your python script add a few lines of Twython to connect to that specific account / what message you want to tweet / who it is mentioning / and the source location of the photo. Make sure the Canon is overwriting the same location in the computer for consistency, make sure you are shooting in JPEG/ and make sure you are saving it to the computer. The Canon can be fired either by wiring a standard 2.5mm jack from the camera to the Pi with GPIO and pulsating on or off. A second usb connection is required to retrieve the files from inside the camera and tweet them from the Pi..
