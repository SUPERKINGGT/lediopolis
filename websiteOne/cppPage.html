Fotoshop

A combination of Instagram and MS Paint for Windows desktops and Tablets written in C++ with QT Framework

Access Github Repository »

Easy touch-friendly UI

Fotoshop is a software designed to bridge the gap between low end paint programs like MS Paint and higher end photo suites like Adobe Photoshop. Fotoshop allows users to open an image, or start with a blank slate. The Image can then be manipulated with traditional dark room settings such as brightness, exposure, lightness, hue, saturation, etc..

Practical paint application

The toolbox holds the brush, pen, bucket, sampler, and eraser tools. The transform area allows the user to rotate the image, mirror the image vertically, mirror the image horizontally, zoom in and zoom out.

Powerful Image Manipulation

The Darkroom tab allows the user to incrementally change exposure, adjust white balance, resize an image, crop an image, and tweak saturation. The filter tab allows users to stylize their photos with seven different filters. The filters included are greyscale, sepia tone, pencil sketch, edge detection, contrast boost, film negative and pixelation. The third tab is used for custom color selection to be used with the toolbox. Color choosing is implemented with three sliders, corresponding with red, green, and blue values as well as a preview window. This allows for precision and customization over a tool like the MS Paint color swatch.
